MailBait is a phishing awareness training simulator developed as a BCA final year project.
The system simulates phishing campaigns in a controlled environment to educate users
about phishing attacks and email security.

## Features
- Admin login
- Campaign creation
- Phishing email simulation
- User click tracking
- Awareness training page
- Campaign statistics dashboard

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
- SMTP (Email)
- AWS EC2 (Ubuntu)

## Disclaimer
This project is developed strictly for educational and awareness purposes.
